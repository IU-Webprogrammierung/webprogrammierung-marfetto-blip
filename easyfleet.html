Easy Fleet

Easy Fleet is a fleet management software developed for Volkswagen to support the booking, maintenance, management, and defect documentation of test vehicles. The system addresses the specific requirements of various user groups, enabling efficient vehicle reservations, comprehensive defect tracking, and real-time information exchange. Features include automated maintenance scheduling and customizable reports to improve operational workflows and ensure transparent documentation throughout the vehicle testing process.
