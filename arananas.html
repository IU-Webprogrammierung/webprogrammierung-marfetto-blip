Arananas

Arananas is a captivating UX demonstration showcasing beautiful animations on a product page. The name is an Italian wordplay blending "Ananas" (pineapple) and "Arancia" (orange), reflecting the smooth transition between these fruits through elegant animations. The project features the fictional brand "Fruttina," which sells beverage cans, highlighting seamless and engaging user interactions.
